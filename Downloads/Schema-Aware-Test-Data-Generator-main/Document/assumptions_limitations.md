# Assumptions & Limitations — v2

## Assumptions

1. **SQL DDL only for parsing** — The DDL parser handles standard SQL (MySQL/PostgreSQL-style). JSON schema and CSV headers are accepted in the UI but converted to DDL format before processing.

2. **AI provider is optional** — The system degrades gracefully to Faker-only mode if no API key is configured.

3. **Gemini 1.5 Flash by default** — Chosen for its free tier (1M tokens/day) and fast response time.

4. **Pool size of 20 AI values per column** — Sufficient for up to 20 unique rows. For larger datasets, the pool is sampled with replacement.

5. **Single FK column per foreign key** — Composite foreign keys are not supported by the generator.

6. **DDL parser is regex-based** — Handles common patterns but may not parse all edge cases (e.g., conditional expressions in DEFAULT, custom types).

---

## Limitations

### AI Schema Regeneration
- AI response quality depends on the configured model. GPT-4o produces better schemas than Llama3-8B.
- The AI may not always produce perfectly valid DDL on the first try. One self-repair attempt is made.
- Very domain-specific requirements (e.g., healthcare-specific table naming conventions) need explicit user instructions.

### Hybrid Generator
- AI columns are detected by **name pattern only** — a column named `notes` will use Faker even if it should have clinical notes, unless the column name matches a known pattern.
- AI value pools are cached **within a single server run**. Restarting the server clears the cache.
- Maximum pool size is 20 values. For datasets > 20 rows, values repeat (with random selection).

### Parser
- Supports `CREATE TABLE` DDL only — no `ALTER TABLE`, triggers, stored procedures, or views.
- Cyclic foreign key relationships (A → B → A) cause infinite loops in the topological sort.
- `ENUM` types are not supported — treated as VARCHAR.

### Performance
- AI calls add 1–30 seconds per column type depending on the provider.
- With 5 AI columns × 1 API call each = approximately 5–15 seconds added to generation time.
- Gemini free tier: 15 requests/minute. Large schemas with many AI columns may hit rate limits.

### Frontend
- Route tree (`routeTree.gen.ts`) is manually maintained — normally generated by TanStack Router CLI. If new routes are added, this file must be updated manually.
- No authentication — intended for local/demo use only.
